Create a "settings/02 - vscode/" folder with:
- settings.json   (your VS Code user settings)
- keybindings.json (optional keyboard shortcuts)
- extensions.txt   (one extension ID per line, e.g. "ms-python.python")

Run: gitmap install vscode-settings
Target: %APPDATA%\Code\User (Windows) or ~/.config/Code/User (Linux)
