# gitmap docs

Open the gitmap documentation website in the default browser.

## Alias

`d`

## Usage

```
gitmap docs
gitmap d
```

## Flags

_No flags._

## Prerequisites

- A default browser configured on your system.

## Examples

```
$ gitmap docs
  ✓ Opened https://gitmap.dev/docs
```

```
$ gitmap d
  ✓ Opened https://gitmap.dev/docs
```

## See Also

- `gitmap help` — show CLI usage summary
- `gitmap version` — show installed version
