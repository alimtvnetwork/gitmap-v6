# gitmap version

Show the current gitmap version number.

## Alias

v

## Usage

    gitmap version

## Flags

None.

## Prerequisites

- None

## Examples

### Example 1: Show version

    gitmap version

**Output:**

    gitmap v2.22.0
    Built:  2025-03-10
    Go:     go1.22.1
    OS:     windows/amd64

### Example 2: Using alias

    gitmap v

**Output:**

    gitmap v2.22.0
